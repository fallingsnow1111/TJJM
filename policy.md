# STIRPAT-EEMD-GRU情景预测论文思路

## 1. 研究目标与总体思路

本文以中国省级面板数据为基础，构建“STIRPAT解释模型 + 残差序列深度学习修正 + 政策情景外推”的碳排放预测框架，用于分析1990-2035年全国及省级碳排放演化路径，并比较不同政策情景下的达峰时间与排放水平差异。

整体逻辑如下：

1. 先基于历史省级数据构建统一面板，剔除西藏样本，避免因基础数据缺失造成口径不一致。
2. 使用STIRPAT模型刻画人口、经济、产业结构、城镇化和能源强度等因素对能源消费的结构性影响。
3. 使用EEMD-GRU对STIRPAT残差进行时序修正，提高非线性波动和省际异质性的拟合能力。
4. 在2024-2035年设置政策情景路径，外推各省能源消费、能源结构和碳排放。
5. 汇总省级预测结果得到全国路径，并识别各情景下的碳达峰年份和峰值水平。

## 2. 模型变量设置

当前版本已删除交通相关变量，不再使用`HighwayMileage`和`PrivateCars`。模型核心变量如下：

| 变量 | 含义 | 作用 |
|---|---|---|
| `Population` | 人口规模 | 反映活动规模效应 |
| `pGDP` | 人均GDP | 反映经济发展水平 |
| `Industry` | 第二产业占比 | 反映产业结构偏重程度 |
| `Urbanization` | 城镇化率 | 反映城镇化发展阶段 |
| `EnergyIntensity` | 能源强度 | 反映单位经济产出的能源消耗压力 |
| `Energy` | 能源消费总量 | 情景预测中的核心中间变量 |
| `CoalShare` | 煤炭消费占比 | 决定化石能源结构和碳排放强度 |
| `OilShare` | 石油消费占比 | 与煤炭、天然气共同决定化石能源结构 |
| `GasShare` | 天然气消费占比 | 体现过渡能源变化 |
| `NonFossilShare` | 非化石能源占比 | 由化石能源占比归一化得到 |
| `CarbonIntensity` | 碳排放强度 | 情景路径中的减排技术和结构综合指标 |

碳排放计算采用能源消费总量与能源结构相结合的方式。煤炭、石油和天然气占比决定单位能源消费对应的碳排放系数，非化石能源占比通过压缩化石能源结构间接降低排放。

## 3. 情景设定

本文设置四类情景，分别对应政策推进力度和经济增长模式的差异。

| 情景 | 中文含义 | 核心设定 |
|---|---|---|
| `baseline` | 基准情景 | 按既有政策与历史趋势稳步推进，2030年达峰 |
| `low_carbon` | 低碳转型情景 | 更早强化节能降碳和产业结构优化，2029年达峰 |
| `green_growth` | 绿色增长情景 | 在较高经济增长下同步加强绿色约束，2030年达峰但峰值低于基准 |
| `extensive` | 粗放发展/转型失败情景 | 能源消费保持较快增长，能源结构和碳强度改善有限，2035年前未达峰 |

这四类情景的论文解释重点不是“预测唯一未来”，而是刻画不同政策路径下的碳排放风险边界：低碳情景代表提前转型收益，绿色增长情景代表增长与减排协同，基准情景代表政策延续路径，粗放情景代表转型不足带来的高排放风险。

## 4. 当前预测结果

最新运行结果如下：

| 情景 | 达峰年份 | 峰值碳排放 | 是否预测期内达峰 | 解释 |
|---|---:|---:|---|---|
| `low_carbon` | 2029 | 924522.89 | 是 | 提前达峰，2030年后快速下降 |
| `baseline` | 2030 | 945329.70 | 是 | 符合2030年前后达峰的政策延续路径 |
| `green_growth` | 2030 | 899803.79 | 是 | 经济增长较高，但绿色约束抵消排放压力，峰值最低 |
| `extensive` | 2035 | 1002044.83 | 否 | 2035年仍处于上升趋势，表示预测期内未达峰 |

从结果看，政策力度差异主要体现在两个方面：

1. 达峰时间：低碳情景最早，基准和绿色增长情景在2030年达峰，粗放情景未能在2035年前达峰。
2. 峰值水平：绿色增长情景虽然经济增长更高，但由于能源结构优化和碳强度下降更强，峰值低于基准；粗放情景则因能源消费持续扩张，排放水平不断抬升。

## 5. 论文写作建议

建议将结果讨论组织成三层：

1. 全国层面：比较四个情景的排放路径、达峰年份和峰值水平，突出“政策强度决定达峰质量”。
2. 省级层面：分析不同省份达峰时间差异，识别高排放省份、提前达峰省份和滞后达峰省份。
3. 政策层面：讨论能源消费控制、煤炭占比下降、产业结构优化和绿色增长协同对达峰结果的影响。

可以采用这样的论文表述：

> 基准情景下，全国碳排放于2030年达到峰值，说明在现有政策延续条件下可以基本实现2030年前后达峰目标。低碳转型情景将达峰时间提前至2029年，表明更积极的能源结构调整和产业结构优化能够有效压缩峰值窗口。绿色增长情景虽然保持较高经济增速，但通过更强的能源效率提升和非化石能源替代，峰值水平低于基准情景，体现出经济增长与减排约束之间的协同可能。相反，粗放发展情景下碳排放在2035年前仍未达峰，说明若转型力度不足，能源消费扩张将显著推高长期排放风险。

## 7. 当前结论口径

本文当前结果支持以下结论：

1. 中国全国碳排放在基准政策延续下可以于2030年达峰。
2. 低碳转型可将达峰时间提前至2029年，但需要更强的能源消费控制和结构优化。
3. 绿色增长并不必然导致更高排放，只要能源效率和能源结构改善同步推进，可以实现较低峰值。
4. 转型失败情景下，碳排放在2035年前仍未达峰，说明高能源消费增长和弱结构调整会显著增加达峰风险。

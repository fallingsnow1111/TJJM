# STIRPAT-EEMD-GRU情景预测论文思路

## 1. 研究目标与总体思路

本文以中国省级面板数据为基础，构建“STIRPAT解释模型 + 残差序列深度学习修正 + 政策情景外推”的碳排放预测框架，用于分析1990-2035年全国及省级碳排放演化路径，并比较不同政策情景下的达峰时间与排放水平差异。

整体逻辑如下：

1. 先基于历史省级数据构建统一面板，剔除西藏样本，避免因基础数据缺失造成口径不一致。
2. 使用STIRPAT模型刻画人口、经济、产业结构、城镇化和能源强度等因素对能源消费的结构性影响。
3. 使用EEMD-GRU对STIRPAT残差进行时序修正，提高非线性波动和省际异质性的拟合能力。
4. 在2024-2035年设置政策情景路径，外推各省能源消费、能源结构和碳排放。
5. 汇总省级预测结果得到全国路径，并识别各情景下的碳达峰年份和峰值水平。

## 2. 模型变量设置

当前版本已删除交通相关变量，不再使用`HighwayMileage`和`PrivateCars`。模型核心变量如下：

| 变量 | 含义 | 作用 |
|---|---|---|
| `Population` | 人口规模 | 反映活动规模效应 |
| `pGDP` | 人均GDP | 反映经济发展水平 |
| `Industry` | 第二产业占比 | 反映产业结构偏重程度 |
| `Urbanization` | 城镇化率 | 反映城镇化发展阶段 |
| `EnergyIntensity` | 能源强度 | 反映单位经济产出的能源消耗压力 |
| `Energy` | 能源消费总量 | 情景预测中的核心中间变量 |
| `CoalShare` | 煤炭消费占比 | 决定化石能源结构和碳排放强度 |
| `OilShare` | 石油消费占比 | 与煤炭、天然气共同决定化石能源结构 |
| `GasShare` | 天然气消费占比 | 体现过渡能源变化 |
| `NonFossilShare` | 非化石能源占比 | 由化石能源占比归一化得到 |
| `CarbonIntensity` | 碳排放强度 | 情景路径中的减排技术和结构综合指标 |

碳排放计算采用能源消费总量与能源结构相结合的方式。煤炭、石油和天然气占比决定单位能源消费对应的碳排放系数，非化石能源占比通过压缩化石能源结构间接降低排放。

## 3. 情景设定

### 3.1 政策依据

情景设定不是对模型参数的随意调整，而是以国家中长期发展目标和能源低碳转型政策为依据，对2024-2035年关键驱动变量进行分阶段外推。本文主要参考以下政策文件：

1. 《中华人民共和国国民经济和社会发展第十四个五年规划和2035年远景目标纲要》：为经济增长、产业结构优化、城镇化推进和2035年发展目标提供总体依据。
2. 《2030年前碳达峰行动方案》：为2030年前后达峰约束、碳强度持续下降、重点领域节能降碳提供直接政策依据。
3. 《“十四五”现代能源体系规划》：为能源消费增速控制、煤炭占比下降、天然气稳步替代和非化石能源比重提升提供依据。
4. 《国家人口发展规划（2016-2030年）》及人口发展趋势判断：为人口增速放缓、2030年前后人口进入平台期提供依据。
5. 《“十四五”推进新型城镇化实施方案》及相关城镇化政策：为城镇化率继续提升但增速逐步放缓提供依据。
6. “十四五”产业结构升级和制造业高质量发展相关部署：为第二产业占比逐步下降、产业结构持续优化提供依据。

基于上述政策文件，本文将影响碳排放路径的核心变量划分为规模因素、结构因素和技术因素三类：人口规模和人均GDP属于规模因素，第二产业占比、城镇化率和能源结构属于结构因素，能源强度和碳排放强度属于技术因素。基准情景反映现有政策延续下的常规发展路径，其他情景则在基准路径上进行方向明确、幅度可解释的差异化调整。

### 3.2 基准情景设定

基准情景用于刻画“现有政策持续推进、经济社会平稳发展、2030年前后实现碳达峰”的路径。其参数设定遵循“前期延续政策目标、中后期体现增速收敛”的原则，具体如下：

1. 能源消费：依据节能降耗和总量强度双控导向，设定能源消费增速由前期较快增长逐步回落。对应参数为2024-2027年4.70%、2028-2030年3.90%、2031-2035年1.20%。
2. 能源强度：依据技术进步、设备更新和能效提升政策，设定单位产出能耗持续下降。对应参数为2024-2027年-0.16%、2028-2030年-0.64%、2031-2035年-2.24%。
3. 碳排放强度：依据碳达峰行动方案和低碳技术推广方向，设定碳强度持续下降。对应参数为2024-2027年-0.20%、2028-2030年-0.80%、2031-2035年-2.80%。
4. 能源结构：依据现代能源体系规划，设定煤炭占比持续下降、石油占比缓慢回落、天然气占比适度提升。对应参数分别为：煤炭占比-0.20、-0.55、-1.40个百分点/年；石油占比-0.13、-0.10、-0.10个百分点/年；天然气占比+0.10、+0.10、+0.05个百分点/年。
5. 人口规模：依据人口发展规划和人口平台期判断，设定人口增速逐步放缓并在后期转负。对应参数为2024-2027年0.03%、2028-2030年0.00%、2031-2035年-0.05%。
6. 人均GDP：依据“十四五”增长目标和2035年远景目标，设定人均GDP保持中高速增长但逐步回落。对应参数为2024-2027年4.75%、2028-2030年4.50%、2031-2035年4.00%。
7. 第二产业占比：依据产业结构升级和实体经济高质量发展方向，设定第二产业占比持续下降。对应参数为2024-2027年-1.20、2028-2030年-0.80、2031-2035年-0.80个百分点/年。
8. 城镇化率：依据新型城镇化政策，设定城镇化率继续提高，但提升速度逐步放缓。对应参数为2024-2027年+0.70、2028-2030年+0.65、2031-2035年+0.50个百分点/年。

因此，基准情景并非“中性情景”，而是具有明确政策含义的延续情景：它假定我国继续推进节能提效、能源结构优化、产业升级和城镇化建设，在不发生重大外部冲击的情况下，于2030年前后实现碳达峰。

### 3.3 其他情景设定

在基准情景基础上，本文进一步设置低碳转型情景、绿色增长情景和粗放发展情景，以比较不同政策强度和发展模式下的碳排放路径差异。其调整遵循“只改变少数关键驱动因素，其余变量沿基准路径延续”的原则。

| 情景 | 中文含义 | 相对基准情景的主要调整 | 政策逻辑 |
|---|---|---|---|
| `low_carbon` | 低碳转型情景 | 进一步压低能源消费增速，加快能源强度、碳强度、煤炭占比和第二产业占比下降，提高天然气替代速度 | 对应更强节能降碳约束和更快结构转型，体现“提前转型、提前达峰” |
| `green_growth` | 绿色增长情景 | 在提高人均GDP增速的同时，进一步强化中期能源强度、碳强度和煤炭占比下降力度 | 对应“增长与减排协同”，考察较高增长能否在绿色约束下实现较低峰值 |
| `extensive` | 粗放发展/转型不足情景 | 提高后期能源消费增速，放缓能源强度和碳强度改善，弱化煤炭替代和产业结构优化 | 对应高投入、高能耗、低转型强度路径，刻画2035年前未达峰的风险边界 |

为便于论文写作，其他情景可直接表述为“相对基准情景的调整值”，而无须重复列出全部参数。核心调整如下：

1. 低碳转型情景：能源消费增速相对基准分别下调0.30、0.53和0.30个百分点；能源强度分别再下降0.00、0.27和0.16个百分点；碳排放强度分别再下降0.00、0.33和0.20个百分点；煤炭占比分别再下降0.00、0.15和0.30个百分点；天然气占比分别再提高0.05、0.05和0.05个百分点；第二产业占比分别再下降0.30、0.30和0.30个百分点。
2. 绿色增长情景：人均GDP增速相对基准分别上调0.20、0.20和0.20个百分点；能源消费增速调整为+0.30、-0.20和+0.20个百分点；能源强度分别再下降0.00、1.04和0.00个百分点；碳排放强度分别再下降0.00、1.30和0.00个百分点；煤炭占比分别再下降0.00、0.80和0.00个百分点；第二产业占比分别再下降0.10、0.20和0.20个百分点。
3. 粗放发展情景：能源消费增速相对基准分别调整为-0.30、+0.10和+2.20个百分点；能源强度改善幅度分别减少0.00、0.53和2.18个百分点；碳排放强度改善幅度分别减少0.00、0.67和2.72个百分点；煤炭占比分别额外提高0.20、0.55和1.40个百分点；天然气占比分别下调0.10、0.10和0.05个百分点；第二产业占比分别提高1.00、0.80和0.80个百分点。

这四类情景的论文解释重点不是“预测唯一未来”，而是刻画不同政策路径下的碳排放风险边界：基准情景代表政策延续路径，低碳情景代表提前转型收益，绿色增长情景代表增长与减排协同，粗放情景代表转型不足带来的高排放风险。

## 4. 当前预测结果

最新运行结果如下：

| 情景 | 达峰年份 | 峰值碳排放 | 是否预测期内达峰 | 解释 |
|---|---:|---:|---|---|
| `low_carbon` | 2029 | 924522.89 | 是 | 提前达峰，2030年后快速下降 |
| `baseline` | 2030 | 945329.70 | 是 | 符合2030年前后达峰的政策延续路径 |
| `green_growth` | 2030 | 899803.79 | 是 | 经济增长较高，但绿色约束抵消排放压力，峰值最低 |
| `extensive` | 2035 | 1002044.83 | 否 | 2035年仍处于上升趋势，表示预测期内未达峰 |

从结果看，政策力度差异主要体现在两个方面：

1. 达峰时间：低碳情景最早，基准和绿色增长情景在2030年达峰，粗放情景未能在2035年前达峰。
2. 峰值水平：绿色增长情景虽然经济增长更高，但由于能源结构优化和碳强度下降更强，峰值低于基准；粗放情景则因能源消费持续扩张，排放水平不断抬升。

## 5. 论文写作建议

建议将结果讨论组织成三层：

1. 全国层面：比较四个情景的排放路径、达峰年份和峰值水平，突出“政策强度决定达峰质量”。
2. 省级层面：分析不同省份达峰时间差异，识别高排放省份、提前达峰省份和滞后达峰省份。
3. 政策层面：讨论能源消费控制、煤炭占比下降、产业结构优化和绿色增长协同对达峰结果的影响。

可以采用这样的论文表述：

> 基准情景下，全国碳排放于2030年达到峰值，说明在现有政策延续条件下可以基本实现2030年前后达峰目标。低碳转型情景将达峰时间提前至2029年，表明更积极的能源结构调整和产业结构优化能够有效压缩峰值窗口。绿色增长情景虽然保持较高经济增速，但通过更强的能源效率提升和非化石能源替代，峰值水平低于基准情景，体现出经济增长与减排约束之间的协同可能。相反，粗放发展情景下碳排放在2035年前仍未达峰，说明若转型力度不足，能源消费扩张将显著推高长期排放风险。

## 7. 当前结论口径

本文当前结果支持以下结论：

1. 中国全国碳排放在基准政策延续下可以于2030年达峰。
2. 低碳转型可将达峰时间提前至2029年，但需要更强的能源消费控制和结构优化。
3. 绿色增长并不必然导致更高排放，只要能源效率和能源结构改善同步推进，可以实现较低峰值。
4. 转型失败情景下，碳排放在2035年前仍未达峰，说明高能源消费增长和弱结构调整会显著增加达峰风险。
